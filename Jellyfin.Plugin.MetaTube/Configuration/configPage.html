HenTube

HenTube metadata plugin for Jellyfin/Emby.

General

Optional MetaTube Server URL. Leave blank to use filename metadata only.
Access token for the server, or blank if no token is required.
Automatically create collections by series.
Display community ratings from the metadata source.
Generate online video trailers in strm format.

Tag mapping

One sed-style substitution per line: s/<regex>/<replacement>/[gI]. Rules run in order before date, studio, ignored-tag, or ordinary-tag matching. Use g for all matches, I for case-insensitive matching, & for the complete match, and \1 through \9 for capture groups.

Filename metadata

One value per line. Matching [...] fields become studios instead of tags.
One sed-style delete expression per line: /<regex>/[I]d. For example, /^(1080p|2160p)$/Id ignores either complete tag without regard to case.

Image

Aspect ratio for primary images, set a negative value to use the default.
JPEG compression quality from 0 to 100. Default: 90.

Substitution

One record per line, separated by equal signs. Leave the target blank to delete the source substring.
One record per line, separated by equal signs. Leave the target blank to delete the source genre.
